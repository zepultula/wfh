# 📘 คู่มือการใช้งานระบบ WFH Daily Report (v3.0.0)

ยินดีต้อนรับสู่ระบบรายงานผลการปฏิบัติงานประจำวัน (Work from Home) ระบบนี้ออกแบบมาเพื่อช่วยให้การสื่อสารระหว่างพนักงานและหัวหน้างานเป็นไปอย่างมีประสิทธิภาพและโปร่งใส

---

## 👤 ส่วนที่ 1: สำหรับพนักงาน (Level 0)

### 1.1 การเข้าสู่ระบบ (Login)
- เข้าใช้งานผ่านหน้าแรกโดยกรอก **อีเมลหรือชื่อผู้ใช้** และ **รหัสผ่าน** ที่ได้รับจากผู้ดูแลระบบ
- รองรับการกรอก **username ย่อ** โดยไม่ต้องใส่ `@domain` เช่น กรอก `zepultula` แทน `zepultula@rmutl.ac.th` ได้ทันที
- หากเข้าใช้งานสำเร็จ ระบบจะพานำท่านเข้าสู่หน้าแดชบอร์ดพนักงานโดยอัตโนมัติ

### 1.2 การบันทึกรายงานประจำวัน
- **เพิ่มรายการงาน:** กดปุ่ม "เพิ่มงาน" เพื่อระบุรายละเอียดงานที่ทำในแต่ละวัน
- **สถานะงาน:** เลือกสถานะของแต่ละกิจกรรม (เสร็จสิ้น / กำลังดำเนินการ / รอดำเนินการ)
- **ความคืบหน้า:** ระบุเปอร์เซ็นต์ความคืบหน้าของงาน (0-100%)
- **บันทึกข้อมูล:** ระบบมีระบบ **Auto-save** บันทึกข้อมูลให้อัตโนมัติเมื่อมีการเปลี่ยนแปลง

### 1.3 การระบุปัญหาและอุปสรรค
- ท่านสามารถพิมพ์รายละเอียดปัญหาที่พบในการทำงานในช่อง "ปัญหาและอุปสรรค" เพื่อให้หัวหน้างานรับทราบและช่วยเหลือได้ทันท่วงที

### 1.4 การสื่อสารและการรับข้อเสนอแนะ
- **การดูคอมเมนต์:** หัวหน้างานสามารถเขียนข้อเสนอแนะหรือคำชมเชยได้ในส่วนของคอมเมนต์
- **สถานะสี (Status Tags):**
    - <span style="color: gray;">●</span> **รับทราบ:** หัวหน้างานรับทราบรายงานแล้ว
    - <span style="color: green;">●</span> **ดีมาก:** งานมีคุณภาพดีเยี่ยม
    - <span style="color: red;">●</span> **ต้องแก้ไข:** โปรดตรวจสอบและแก้ไขตามคำแนะนำ
    - <span style="color: orange;">●</span> **ติดตามด่วน:** งานมีความสำคัญที่ต้องติดตามเป็นพิเศษ
- **การตอบกลับ:** ท่านสามารถเขียนคอมเมนต์โต้ตอบกับหัวหน้างานได้ในหน้ารายงานของวันนั้นๆ

### 1.5 การดูประวัติย้อนหลัง
- ท่านสามารถเลือกวันที่จากปฏิทินเพื่อเรียกดูรายงานในอดีตได้ (ข้อมูลย้อนหลังจะเป็นแบบอ่านอย่างเดียว ไม่สามารถแก้ไขได้หลังจากวันที่กำหนด)

## 👨‍💼 ส่วนที่ 2: สำหรับหัวหน้างานและแอดมิน (Level 1-9)

### 2.1 แดชบอร์ดสรุปผล (Supervisor Dashboard)
- **สถิติรวม:** ดูจำนวนพนักงานที่ส่งรายงานแล้ว, ยังไม่ส่ง และพนักงานที่มีปัญหาในวันนี้
- **การจัดกลุ่ม:** ข้อมูลพนักงานจะถูกจัดกลุ่มตาม **หน่วยงาน/แผนก** เพื่อให้ง่ายต่อการตรวจสอบ
- **การกรองข้อมูล (Filters):**
    - **ทั้งหมด:** แสดงพนักงานทุกคนที่รับผิดชอบ
    - **ส่งแล้ว:** รายชื่อพนักงานที่กรอกรายงานเข้ามาแล้ว
    - **มีปัญหา:** พนักงานที่ระบุปัญหาในรายงาน (จะแสดงสัญลักษณ์เตือน)
    - **ยังไม่ส่ง:** รายชื่อพนักงานที่ยังไม่ได้ทำรายงานของวันนี้
- **การดูข้อมูลตามสิทธิ์:**
    - **Level 1-3 (Supervisor):** เห็นเฉพาะลูกน้องในสายบังคับบัญชาของตนเอง
    - **Level 9 / Super Admin:** เห็นพนักงานทุกคนในระบบ

### 2.2 การตรวจรายงานและให้ Feedback
- คลิกปุ่ม **"ดูรายละเอียด / คอมเมนต์"** เพื่อดูรายงานฉบับเต็มของพนักงาน
- อ่านรายละเอียดปัญหา (ถ้ามี)
- พิมพ์คอมเมนต์และเลือก **สถานะแท็ก** เพื่อสื่อสารกลับไปยังพนักงาน

### 2.3 📊 สถิติรายเดือน (สำหรับ Admin ทุกระดับ)
- กดปุ่ม **"📊 สถิติรายเดือน"** ในแถบเมนูด้านบน
- **เลือกเดือน:** ใช้ตัวเลือกเดือน (Month Picker) เพื่อดูสถิติของเดือนที่ต้องการ
- **KPI สรุป:** แสดง 4 ตัวชี้วัดหลัก:
    - จำนวนผู้ใช้ทั้งหมด
    - คน-วันที่ส่งรายงาน
    - Compliance% (อัตราการส่งรายงาน)
    - Avg Progress% (ความคืบหน้าเฉลี่ย)
- **ตารางรายบุคคล:** แสดงรายละเอียดของแต่ละคน จัดกลุ่มตามแผนก:
    - จำนวนวันที่ส่งรายงาน, Compliance%, Avg Progress% (แสดงสีตามเกณฑ์ เขียว/เหลือง/แดง)
    - รูปแบบทำงาน (WFH / On-site / Hybrid)
    - จำนวนงานที่เสร็จ / ทั้งหมด
    - จำนวนวันที่มีปัญหา
- **ส่งออก Excel:** กดปุ่ม **"ดาวน์โหลด Excel"** เพื่อดาวน์โหลดไฟล์ `.xlsx` สรุปสถิติเดือนนั้น
- **ข้อมูลที่แสดงตามสิทธิ์:**
    - **Level 1-3 (Supervisor):** เห็นเฉพาะตนเอง + ลูกน้องในสายบังคับบัญชา
    - **Level 9 / Super Admin:** เห็นพนักงานทุกคนในระบบ

### 2.4 การจัดการผู้ใช้ (เฉพาะ Super Admin - Level 9)
- เข้าถึงหน้าจัดการผ่านเมนู **"⚙ จัดการผู้ใช้"**
- **เพิ่ม/แก้ไข/ลบ:** สามารถจัดการข้อมูลพื้นฐาน, เปลี่ยนรหัสผ่าน และกำหนดระดับสิทธิ์ (Level) ให้พนักงาน
- **ระบบ Ignore (ยกเว้นแสดงผล):** สามารถกดปิดสถานะการแสดงผลสำหรับพนักงานที่ลาออกหรือพักงานชั่วคราวได้ โดยไม่ต้องลบข้อมูลออกจากฐานข้อมูล
- **ค้นหา:** ค้นหาพนักงานตาม ชื่อ, อีเมล หรือหน่วยงาน ได้อย่างรวดเร็ว

### 2.5 การจัดการสายบังคับบัญชา (เฉพาะ Super Admin - Level 9)
- เข้าถึงผ่านเมนู **"🔗 สายบังคับบัญชา"**
- กำหนดว่าหัวหน้าแต่ละคนสามารถเห็นรายงานของลูกน้องคนไหนบ้าง
- **เพิ่มผู้ประเมิน:** ค้นหาชื่อจาก Search แล้วคลิกเลือก
- **จัดลำดับ:** ใช้ปุ่ม ↑↓ เพื่อเรียงลำดับผู้ประเมิน (ลำดับที่ 1 คือผู้ประเมินหลัก)

### 2.6 ⬇ ส่งออกรายงานประจำวันเป็น Excel (สำหรับ Admin ทุกระดับ)
- กดปุ่ม **"⬇ Excel"** ที่อยู่ข้างตัวเลือกวันที่ในหน้า Dashboard
- ระบบจะดาวน์โหลดไฟล์ `.xlsx` สรุปรายงานของวันที่เลือกอย่างละเอียด ประกอบด้วย:
    - ชื่อ-สกุล, ตำแหน่ง, หน่วยงาน
    - รูปแบบทำงาน (WFH/On-site/Hybrid)
    - ความคืบหน้า % (สีเขียว/เหลือง/แดง ตามเกณฑ์)
    - รายการงาน + สถานะทุกรายการ
    - ปัญหา/อุปสรรค, แผนวันพรุ่งนี้
    - คอมเมนต์ทั้งหมด (รวมชื่อผู้เขียน + tag)
    - รายชื่อผู้ยังไม่ส่งรายงาน (สีแดง) จัดกลุ่มตามแผนก
- **ขณะดาวน์โหลด:** ปุ่มจะแสดงไอคอนหมุนและข้อความ "กำลังโหลด..." เมื่อดาวน์โหลดเสร็จจะเปลี่ยนเป็น "✓ สำเร็จ"
- **ข้อมูลตามสิทธิ์:**
    - **Level 1-3 (Supervisor):** ได้เฉพาะตนเอง + ลูกน้องในสายบังคับบัญชา
    - **Level 9 / Super Admin:** ได้พนักงานทุกคนในระบบ

## ⚠️ ส่วนที่ 3: คำแนะนำและการแก้ปัญหา

- **ลืมรหัสผ่าน:** หากยังเข้าสู่ระบบได้ ท่านสามารถเปลี่ยนรหัสผ่านเองได้ผ่านเมนู Profile (คลิกชื่อของท่านที่มุมขวาบน → "เปลี่ยนรหัสผ่าน") หากเข้าสู่ระบบไม่ได้ โปรดติดต่อผู้ดูแลระบบ (Super Admin)
- **ระบบวนลูปหน้า Login:** หากระบบวนกลับมาหน้า Login ซ้ำๆ ให้ตรวจสอบว่าได้อนุญาตให้ Browser เก็บ LocalStorage หรือไม่ หรือลองล้าง Cache ของ Browser แล้วเข้าใช้งานใหม่อีกครั้ง
- **ข้อมูลไม่แสดง:** ตรวจสอบการเชื่อมต่ออินเทอร์เน็ต และตรวจสอบสถานะ "ส่งแล้ว/ยังไม่ส่ง" ในหน้าแดชบอร์ด
- **ไม่เห็นรายงานลูกน้อง:** ตรวจสอบที่ Super Admin ว่าสายบังคับบัญชา (🔗) กำหนดถูกต้องหรือไม่
